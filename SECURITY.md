# Security Policy

## Supported versions

Only the latest release is actively supported with security updates.

| Version | Supported |
|---------|-----------|
| latest  | yes       |

## Reporting a vulnerability

If you discover a security vulnerability, please **do not** open a public issue.

Email the maintainer at **rouven@himmelstein.cloud**. You should receive a response within 48 hours.

Please include:

- A description of the vulnerability
- Steps to reproduce
- Affected version(s)
- Any proposed fix (if available)

We take all reports seriously and will work to confirm and address them promptly.
